<h1 align="center">KnowHub</h1>

<p align="center">
  A knowledge management platform built with <strong>Django</strong> and <strong>React</strong> that allows users to create, search, and manage knowledge entries efficiently.
</p>

<p align="center">
  <a href="#features">Features</a> •
  <a href="#screenshots">Screenshots</a> •
  <a href="#tech-stack">Tech Stack</a> 
</p>

## Features

- **Create and Manage Entries**: Intuitive interface for creating and managing knowledge entries.
- **Search and Filter**: Easily search through entries using keywords and filters.
- **Responsive Design**: Fully responsive design that works well on various screen sizes.

## Tech Stack

### Backend

- **Python**: Developed using Python with Django, providing robust and scalable architecture.
- **Django REST Framework**: Used for creating RESTful APIs consumed by the frontend.

### Frontend

- **React**: The frontend is built using React, offering a dynamic and responsive user interface.
